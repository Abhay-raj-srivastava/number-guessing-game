# 🎮 Number Guessing Game (Java Swing)

A simple interactive **Number Guessing Game** built using **Java Swing GUI**.  
The player has to guess the randomly generated number within limited attempts, and the game provides hints like **Too High** or **Too Low**.

---

## 🚀 Features
- 🎲 Random number generation
- 🖥️ Java Swing GUI
- 🔢 Attempts counter
- 📢 Hint messages (Too High / Too Low)
- ✅ Input validation
- 🛑 Game over handling
- 🎯 Winning message popup

---

## 🛠️ Tech Stack
- **Java**
- **Java Swing**
- **OOP Concepts**
- **IntelliJ IDEA**
- **Git & GitHub**

---

## ▶️ How to Run
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Navigate to `src/Main.java`
4. Run the `main()` method
5. Start guessing 🎉
